# tmuxinator

## Local sources

- `tmuxinator help`
- `tmuxinator help start`
- `tmuxinator help local`
- `tmuxinator doctor`

## Local configs

- `~/.dots/.config/tmuxinator/*.yml` (stowed to `~/.config/tmuxinator`)
- `~/.dots/no_stow/.zsh_aliases_public` (`alias mux='tmuxinator'`)
- `~/.dots/.zshrc` (Oh My Zsh `tmuxinator` plugin)

## Official sources

- Upstream repository: `https://github.com/tmuxinator/tmuxinator`
- Upstream documentation: `https://github.com/tmuxinator/tmuxinator#readme`

## Local gotchas

- `~/.config/tmuxinator` is a symlink to `~/.dots/.config/tmuxinator`; inspect/edit the dotfiles path for source-controlled layouts.
